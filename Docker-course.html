🐳 DockerHub

مرحباً بك في دورة Docker التفاعلية

Docker أداة لتشغيل التطبيقات داخل حاويات خفيفة ومعزولة. في هذه الدورة ستتعلم الأساسيات مع أمثلة عملية ومحاكي أوامر.

لماذا Docker؟

  • 🐞 بيئة متطابقة في كل مكان (جهازك، السيرفر، السحابة).
  • ⚡ تشغيل وإيقاف التطبيقات بسرعة.
  • 📦 عزل الاعتماديات دون تعارض.

أوامر Docker الأساسية

# تشغيل حاوية ngnix
docker run -d -p 8080:80 --name myweb nginx:alpine

# عرض الحاويات النشطة
docker ps

# إيقاف الحاوية
docker stop myweb

# حذف الحاوية
docker rm myweb

# بناء صورة
docker build -t myapp:1.0 .

⚡ محاكي أوامر Docker

جرّب كتابة أوامر مثل docker run hello-world أو docker ps

أهلاً بك في محاكي Docker الطرفي 🐳 اكتب `help` لعرض الأوامر المتاحة
$

تمرين 1: كتابة Dockerfile

اكتب Dockerfile لتطبيق Python بسيط يستمع على port 5000.

FROM python:3.10-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
EXPOSE 5000
CMD ["python", "app.py"]

تمرين 2: docker-compose

ارفع تطبيق ويب مع قاعدة بيانات Postgres باستخدام compose.

version: '3'
services:
  web:
    build: .
    ports:
      - "8000:5000"
  db:
    image: postgres:15-alpine
    environment:
      POSTGRES_PASSWORD: secret
